# Book Search Servlet Project

## Description
This project is a Java Servlet-based web application that allows users to search books by author name using a database.

## Technologies Used
- Java Servlet
- JDBC
- Supabase (PostgreSQL)
- HTML

## Features
- Search books using author name
- Displays book title, author, and price
- Uses database connectivity with Supabase

## How to Run
1. Deploy the project on Apache Tomcat server
2. Open index.html in browser
3. Enter author name to search
